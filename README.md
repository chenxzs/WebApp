# 技术栈
Node+express+Vue全家桶+axios+webpack

## 说明
项目分为两部分 <br/>
Server :node+express，前端需要的数据全部放在Server->data文件夹里面<br/>
web:CSS3+H5+Vue.js

## 启动
1. cd至webapp>server目录下 <br>
   npm install  安装依赖 <br/>
   npm run start 启动服务端 <br/>

2. cd至webapp>web目录下 <br/>
   npm install  安装依赖 <br/>
   npm run serve 启动页面 <br/>
