# 技术栈
Node+express+Vue全家桶+axios+webpack

## 说明
      通过媒体查询页面能自适应不同设备，项目分为两部分 
      Server :node+express，前端需要的数据全部放在Server->data文件夹里面
      web:CSS3+H5+Vue.js

## 启动
1. cd至webapp>server目录下 
      npm install  安装依赖 
      npm run start 启动服务端 

2. cd至webapp>web目录下
   npm install  安装依赖 
   npm run serve 启动页面
